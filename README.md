
## MongoLara Config

<p>1. define MONGODB_URI = where mongo db is running </p>
<p>2. define MONGODB_DATABASE =  mongo db data base name </p>

## MongoLara instalation 

<p>$ cd mongolara</p>
<p>$ composer update </p>
<p>$ php artisan migrate </p>
<p>$ php artisan serve </p>

## Front instalation 

<p>$ cd front </p>
<p>$ npm install </p>
